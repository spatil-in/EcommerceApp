# React Redux Ecommerce Website

A fully responsive Ecommerce web application built using React.js and Redux Toolkit.  
The project allows users to browse products, add items to cart, manage quantities, and experience smooth navigation with modern UI design.

---

## 🚀 Features

- Product Listing
- Add to Cart
- Remove from Cart
- Update Cart Quantity
- Redux Toolkit State Management
- Responsive Design
- Product Search & Filter
- React Router Navigation
- Local Storage Data Persistence
- Fast Performance with Vite

---

## 🛠️ Tech Stack

- React.js
- Redux Toolkit
- React Redux
- React Router DOM
- JavaScript (ES6+)
- HTML5
- CSS3 / Tailwind CSS
- Vite
